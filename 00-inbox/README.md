# 00 Inbox

> **用途**：临时收集箱。存放碎片想法、待整理笔记、从外部剪贴的内容、随手记录的问题和灵感。

---

## 使用规则

1. **先扔进来**：任何还来不及分类的内容，先放到这里，不要纠结放在哪个领域。
2. **定期清理**：每周至少清理一次，把 inbox 中的内容移动到对应领域，或删除无效内容。
3. **不长期停留**：inbox 不是终点站，笔记在这里停留不超过两周。
4. **命名随意**：inbox 中的文件不需要严格命名规范，但建议文件名能表达主题。

---

## 清理流程

```
inbox 中的笔记
    ├── 有价值 → 移动到 01-08 对应领域
    │              └── 使用 _templates/ 中的模板重写
    ├── 待确认 → 保留在 inbox，标注 #待整理 标签
    └── 无价值 → 删除或移入 _archive/
```

---

## 常用标签

- `#待整理`：需要进一步分类和完善的笔记
- `#灵感`：临时想法，待验证
- `#剪贴`：从外部复制的内容，需要消化
- `#问题`：工作中遇到的问题，待研究

---

> **提示**：inbox 空了不代表你没有想法，而是说明你的知识库在高效运转。
