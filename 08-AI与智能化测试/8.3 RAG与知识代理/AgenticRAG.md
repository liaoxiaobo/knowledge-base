# AgenticRAG

## 一句话本质

Agentic RAG 是在传统 RAG 基础上引入 Agent 的推理与工具调用能力，让检索过程可以动态迭代、多源验证，提升复杂问答的准确性。

## 核心内容

1. **传统 RAG 局限**：单次检索容易遗漏或召回无关内容。
2. **Agentic 改进**：检索器作为工具，Agent 可多次调用、比较来源、生成查询改写。
3. **查询分解**：复杂问题拆成多个子查询分别检索再综合。
4. **来源验证**：Agent 对检索结果进行可信度评估和交叉验证。
5. **测试关注点**：检索策略、推理链正确性、答案 grounding 程度。
