# AI辅助UI测试

## 一句话本质

AI 辅助 UI 测试是利用大模型理解页面结构、生成定位表达式或断言，降低 UI 自动化脚本编写和维护成本。

## 核心内容

1. **元素定位生成**：从截图或 DOM 生成稳定的 role/testid 选择器。
2. **用例生成**：基于自然语言描述或用户操作录制生成测试脚本。
3. **脚本自愈**：页面变更后自动修复失效的定位表达式。
4. **视觉测试**：结合 AI 进行截图差异的智能判断。
5. **测试重点**：生成结果的稳定性、可维护性、边界场景覆盖。
