# Agent架构设计

## 一句话本质

Agent 架构设计是围绕 LLM 核心能力，合理编排规划、记忆、工具、行动和反馈模块，构建可扩展、可观测的自主系统。

## 核心内容

1. **核心组件**：LLM 大脑、规划模块、记忆模块、工具集、执行环境。
2. **单 Agent vs 多 Agent**：单 Agent 适合任务闭环，多 Agent 适合角色分工。
3. **记忆设计**：短期工作记忆、长期向量记忆、事实记忆。
4. **观测与日志**：关键节点记录 Thought、Action、Observation，便于调试。
5. **容错与回退**：工具失败、模型幻觉时的降级策略。
