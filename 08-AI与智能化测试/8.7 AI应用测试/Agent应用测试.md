# Agent应用测试

## 一句话本质

Agent 应用测试是验证自主 Agent 在目标理解、规划执行、工具调用、多轮交互和异常处理上的正确性与稳定性。

## 核心内容

1. **规划测试**：复杂目标能否拆分为合理步骤。
2. **工具调用测试**：参数正确、调用顺序、错误恢复。
3. **多轮交互测试**：上下文记忆、用户澄清、动态调整。
4. **边界测试**：工具不可用、超时、模型幻觉时的行为。
5. **可观测性**：Thought-Action-Observation 链路完整记录。
