# ClaudeCodeSkill与MCP扩展

## 一句话本质

Claude Code 通过 Skill 扩展内置能力，并通过 MCP 协议连接外部 Server，让 AI 助手能使用团队私有工具和知识源。

## 核心内容

1. **Claude Code Skill**：定义在 .claude/skills/ 下的 YAML/Markdown 能力单元。
2. **MCP Server 集成**：在 settings.json 中配置 mcpServers，连接外部工具。
3. **典型扩展场景**：自定义代码审查、内部平台操作、知识库查询。
4. **权限控制**：区分哪些 Skill/MCP 工具可在自动模式下使用。
5. **测试方法**：验证 Skill 提示词效果、MCP 工具返回格式。
