## 七、关键边界提醒

- **3.1.1 Pytest vs 2.1.2 Python 测试编程实践**：3.1.1 讲 Pytest 框架专属能力（fixture、parametrize、hook、插件），2.1.2 讲 Python 语言级能力（mock、解析、并发、装饰器）。
- **3.2.4 vs 3.2.5**：3.2.4 讲"怎么设计框架"，3.2.5 讲"怎么评判和优化框架"。
- **通用方法与项目案例**：同一主题只保留一份通用方法论，项目细节通过 `#项目` 标签横向串联。
