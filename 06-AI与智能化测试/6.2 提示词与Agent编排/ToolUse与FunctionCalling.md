# ToolUse与FunctionCalling

## 一句话本质

Tool Use / Function Calling 是让大模型在生成回复时识别需要调用的外部工具或函数，并将结构化参数返回给系统执行，从而扩展模型的能力边界。

## 核心内容

1. **核心机制**：模型通过特殊 token 或 JSON 格式输出函数调用意图，由外部执行后返回结果。
2. **典型流程**：用户提问 → 模型判断需要工具 → 生成调用参数 → 系统执行 → 结果回传 → 模型生成最终回复。
3. **常见工具类型**：搜索、数据库查询、计算、API 调用、代码执行。
4. **测试重点**：参数正确性、调用时序、错误处理、工具返回对生成结果的影响。
5. **安全注意**：防止提示词注入导致非授权工具调用。
