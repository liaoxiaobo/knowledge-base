# Agent工作流设计

## 一句话本质

Agent 工作流设计是把复杂任务拆分为规划、执行、观测、反思的循环步骤，让 LLM 能够自主调用工具并完成多步目标。

## 核心内容

1. **ReAct 模式**：Thought（思考）→ Action（行动）→ Observation（观测）循环。
2. **规划能力**：将用户目标拆解为可执行的子任务列表。
3. **工具编排**：根据任务动态选择合适工具并传递上下文。
4. **记忆管理**：短期上下文记忆与长期知识检索结合。
5. **终止条件**：明确任务完成或失败的判断标准。
