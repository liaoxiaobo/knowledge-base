# 多Agent协作

## 一句话本质

多 Agent 协作是把复杂任务拆分为多个角色专精的 Agent，通过对话、任务委派或共享状态协同完成单个 Agent 难以胜任的目标。

## 核心内容

1. **协作模式**：主管-工人、平等协商、流水线、竞争投票。
2. **角色划分**：按功能（规划、执行、审查）或按领域（前端、后端、测试）拆分。
3. **通信机制**：消息传递、共享内存、黑板系统。
4. **冲突解决**：当 Agent 意见不一致时的仲裁策略。
5. **测试重点**：协作流程正确性、信息不丢失、任务不重复。
