# Skill设计规范

## 一句话本质

Skill 是 Claude Code 等 AI 编程工具中可复用的能力单元，设计规范关注输入输出、提示词结构、工具调用和错误处理的标准化。

## 核心内容

1. **Skill 组成**：名称、描述、触发条件、提示词模板、可用工具。
2. **原子性原则**：一个 Skill 聚焦一个明确任务，避免过度复杂。
3. **上下文管理**：清晰定义 Skill 需要接收和返回的上下文字段。
4. **可测试性**：Skill 的输入输出应可独立验证。
5. **版本治理**：Skill 接口变更需向后兼容或明确迁移路径。
